# JavaScript-Source

## What is JavaScript-Source 
JavaScript source is the original JavaScript code written by a developer that runs in a web browser.<br>
Example:<br>
```JavaScript
function greet() {
    alert("Hello!");
}

greet();
```

When you visit a website and see files like script.js or code inside <script> tags, that is the JavaScript source.
